# Coaching Automation System

Automates WhatsApp notifications for coaching centers:
- Attendance alerts
- Fee reminders
- Exam updates

## Tech Stack
- .NET Web API
- Google Sheets API
- Twilio WhatsApp API

## Setup

1. Clone repo
2. Add `appsettings.json`
3. Add `credentials.json`
4. Run:
   dotnet run

## API

GET /api/student/run
